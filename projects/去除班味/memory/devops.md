- 日志收集: 阿里云SLS
- 告警通知: 钉钉群

**备份策略**:
- RDS自动备份（30天保留）
- OSS跨区域复制
- 额外pg_dump逻辑备份
- RTO目标4小时，RPO目标1小时

## 重要决策

### [2026-04-30] 选择阿里云作为云服务商
**原因**:
- 微信小程序生态天然配合
- OSS/COS与CDN深度集成
- 云监控、ARMS、SLS一站式服务
- 完善的云安全体系

### [2026-04-30] 采用Docker容器化部署
**原因**:
- 环境一致性（开发/测试/生产）
- 快速回滚能力
- 资源隔离
- 便于水平扩展

### [2026-04-30] 使用GitHub Actions CI/CD
**原因**:
- 与代码仓库原生集成
- 免费额度足够MVP使用
- 社区生态丰富
- 无需额外CI服务器
