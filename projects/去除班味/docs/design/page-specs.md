# 去除班味 — 页面设计规范

## 1. 首页/检测入口

### 布局
- 全屏页面，无底部Tab（进入后显示Tab）
- 顶部：品牌Logo + 动态文案（每日变化）
- 中部：大圆形拍照按钮（核心CTA，占视觉中心）
- 下方：最近检测记录（横向滑动卡片）
- 底部：简要功能说明（3个icon + 文字）

### 配色
- 背景: 渐变 --color-surface → --color-brand-muted (微妙暖色渐变)
- 主按钮: --color-brand，带脉冲动画光环
- 文字: --color-text-primary / --color-text-secondary

### 字体
- 顶部欢迎语: --text-display, 站酷快乐体
- 主按钮文字: --text-heading, 500 weight
- 功能说明: --text-small

### 交互
- 拍照按钮: 长按开始脉冲动画，松手进入拍照页
- 最近记录: 左滑查看历史，点击直接进入报告页
- 页面入场: 元素从下方 stagger 入场，间隔 80ms

### 核心元素
```
拍照按钮:
- 直径: 120px (含光环 160px)
- 背景: --color-brand
- 图标: 📷 或相机图标
- 外圈: 2px dashed --color-brand-light，旋转动画
- 内圈: 实心，带阴影 --shadow-lg
- 按下: scale(0.95)
```

## 2. 拍照/上传页

### 布局
- 顶部导航栏（返回 + 标题"检测班味"）
- 中部：场景选择器（横向标签）
  - 工位检测 / 自拍检测 / 桌面检测 / 任意照片
- 下方：拍照区域 / 相册选择
- 底部：拍照按钮（大圆形）+ 切换摄像头

### 配色
- 背景: --color-surface
- 场景标签: 未选中 ghost，选中 --color-brand 背景
- 拍照按钮: --color-brand，白色图标

### 字体
- 标题: --text-heading
- 场景标签: --text-body
- 提示文字: --text-small, --color-text-tertiary

### 交互
- 场景切换: 滑动切换，标签底部有小滑块指示器
- 拍照: 按下按钮缩小，快门闪光效果（全屏白色渐变消失）
- 相册选择: 网格展示最近照片，点击选中带 --color-brand 边框

### 场景差异
```
工位检测: 提示"对准你的工位，越乱越真实"
自拍检测: 提示"露出你的'上班脸'" + 人脸框
桌面检测: 提示"展示你的办公桌/电脑桌面"
任意照片: 提示"任何你觉得有班味的照片"
```

## 3. 分析中页

### 布局
- 全屏居中内容
- 顶部：动态文案（"正在扫描班味分子..."等，轮播）
- 中部：动画图形（SVG或CSS动画）
- 下方：进度条 + 趣味事实

### 配色
- 背景: 根据预估浓度渐变的动态背景（从浅色逐渐变深）
- 动画元素: --color-brand
- 进度条: 情绪色渐变

### 字体
- 动态文案: --text-display, 站酷快乐体
- 趣味事实: --text-body
- 进度百分比: --text-heading

### 交互
- 文案轮播: 每2秒切换一条幽默文案（"正在分析咖啡渍浓度..."）
- 进度条: 非线性增长，快慢交替增加趣味性
- 动画元素: 漂浮的办公元素（☕ 📎 📝）

### 趣味文案库
```
- "正在扫描班味分子..."
- "检测咖啡依赖指数..."
- "分析工位混乱程度..."
- "计算外卖盒堆积高度..."
- "读取眼神空洞指数..."
- "统计未读消息数量..."
- "测量肩膀僵硬程度..."
```

## 4. 检测报告页（核心页面）

### 布局
- 顶部：返回按钮 + 分享按钮
- 上部：总分大数字 + 评级标签
- 中部：雷达图（6个维度）
- 下部：辣评文案（带引号装饰）
- 底部：识别元素列表 + 去班味方案按钮

### 配色
- 背景: 情绪色背景（根据分数）
- 总分数字: 情绪色，超大字号 --text-score
- 雷达图: 情绪色填充
- 辣评卡片: 白色背景，左侧装饰引号

### 字体
- 总分: --text-score (64px), 站酷快乐体
- 评级: --text-display, 大写风格
- 辣评: --text-heading, 霞鹜文楷（增加人文感）
- 维度标签: --text-small

### 交互
- 页面入场: 总分数字从0滚动到实际分数（计数动画）
- 雷达图: 从中心向外展开动画（600ms ease-out-expo）
- 辣评: 打字机效果逐字出现
- 分享按钮: 点击展开分享选项（海报/链接）
- 下滑: 查看更多详情

### 分数评级映射
```
0-30分:  "清新脱俗" → --color-fresh → "你是职场小白莲，继续保持！"
31-60分: "微醺班味" → --color-tipsy → "班味初现，还有救！"
61-85分: "班味浓郁" → --color-strong → "班味缠身，需要紧急干预！"
86-100分:"腌入味了" → --color-extreme → "恭喜你，已经是班味的形状了"
```

### 雷达图维度
```
1. 眼神空洞度    2. 工位混乱度
3. 咖啡依赖度    4. 外卖堆积度
5. 肩膀僵硬度    6. 消息未读数
```

### 识别元素展示
```
卡片式列表:
- 元素图标 + 名称 + 识别置信度
- 例: ☕ 咖啡杯 x3 (置信度 99%)
- 例: 🥡 外卖盒 x2 (置信度 87%)
- 例: 📎 回形针散落 (置信度 92%)
```

## 5. 去班味方案页

### 布局
- 顶部：返回 + 标题"去班味方案"
- 上部：方案总览（"为你推荐X个方案"）
- 中部：方案卡片列表（纵向滚动）
- 底部：收藏/分享按钮

### 配色
- 背景: --color-surface
- 方案卡片: 每个卡片不同浅色背景（薄荷绿/淡紫/暖黄）
- 重要度标签: 情绪色

### 字体
- 标题: --text-heading
- 方案标题: --text-subheading, 500 weight
- 方案描述: --text-body
- 步骤编号: --text-display, 站酷快乐体

### 交互
- 卡片展开: 点击展开详情（grid-template-rows 动画）
- 收藏: 心形图标切换，带缩放弹跳动画
- 完成打卡: 点击"完成"打勾，卡片变灰并移动到底部

### 方案卡片结构
```
┌─────────────────────────────┐
│ 🔥 紧急  方案名称            │
├─────────────────────────────┤
│ 简介描述文字...              │
│                             │
│ 步骤:                       │
│ ① 第一步                    │
│ ② 第二步                    │
│                             │
│ [☑️ 已完成]  [收藏❤️]        │
└─────────────────────────────┘
```

### 示例方案
```
1. "工位断舍离" - 清理桌面，只留必需品
2. "15分钟工间操" - 缓解肩膀僵硬
3. "咖啡戒断计划" - 逐步减少咖啡因摄入
4. "窗边远眺疗法" - 每2小时看远处5分钟
5. "准点下班挑战" - 今天不加班！
```

## 6. 圈子/社区页

### 布局
- 顶部：标题"班味圈子" + 发布按钮
- 中部：Tab切换（"推荐" / "附近" / "我的"）
- 下部：信息流卡片列表
- 底部：Tab栏

### 配色
- 背景: --color-surface
- 卡片: --color-surface-raised
- 浓度标签: 对应情绪色
- 点赞/评论: --color-text-tertiary（激活时 --color-brand）

### 字体
- 页面标题: --text-display, 站酷快乐体
- 用户名: --text-body, 500 weight
- 内容: --text-body
- 时间戳: --text-caption, --color-text-tertiary

### 交互
- 下拉刷新: 自定义动画（班味分子飘散）
- 点赞: 心形缩放动画 + 数字递增
- 评论: 底部弹出评论框

## 7. 分享海报页

### 布局
- 顶部：返回 + 标题"生成海报"
- 中部：海报预览区（正方形，适合朋友圈）
- 下方：样式切换（3-4种模板）
- 底部：保存/分享按钮

### 配色
- 海报模板根据浓度使用不同配色方案
- 每种模板有独特的装饰元素

### 字体
- 海报标题: --text-display, 站酷快乐体
- 分数: --text-score
- 辣评: --text-heading, 霞鹜文楷
- 二维码区域: --text-caption

### 交互
- 模板切换: 滑动切换，预览区实时更新
- 长按海报: 放大预览
- 保存: 下载到本地 + 成功提示

### 海报模板设计

#### 模板A "Office档案"（复古风）
```
- 背景: 米黄色纸张纹理
- 边框: 复古双线边框
- 标题: "班味检测报告"
- 元素: 打字机效果数字、印章装饰
- 风格: 90年代Office文档
```

#### 模板B "诊断书"（戏谑风）
```
- 背景: 白色医生处方单风格
- 标题: "班味诊断书"
- 元素: 红十字、手写体签名
- 风格: 医院诊断单戏仿
```

#### 模板C "潮流杂志"（时尚风）
```
- 背景: 高饱和度渐变色
- 标题: 大字排版
- 元素: 几何图形、潮流贴纸
- 风格: 时尚杂志封面
```

#### 模板D "极简艺术"（ins风）
```
- 背景: 纯色
- 排版: 大量留白
- 字体: 超大字号 + 小字注释
- 风格: 极简主义海报
```

## 页面流程图

```
首页（检测入口）
    ↓ 点击拍照
拍照/上传页
    ↓ 确认照片
分析中页（加载动画）
    ↓ 分析完成
检测报告页（核心）
    ↓ 点击方案
去班味方案页
    ↓ 返回 / 去社区
圈子/社区页
    ↓ 分享按钮
分享海报页
```

## 状态与空状态设计

### 首页空状态（无历史记录）
```
- 欢迎插画（🎉 新人专属）
- 文案: "第一次来？快测测你的班味浓度！"
- CTA: 大拍照按钮更加突出
```

### 社区空状态（无内容）
```
- 插画: 空荡荡的工位
- 文案: "这里还没有班味，你是第一个！"
- CTA: "去检测并分享"
```

### 网络错误状态
```
- 插画: 📡 断网
- 文案: "班味信号中断，检查网络后重试"
- CTA: 重试按钮
```

### 加载状态
```
- 骨架屏: 圆角矩形脉冲动画
- 或者: 趣味加载文案轮播
```
