## 一、冷启动策略

### 1.1 种子用户招募（上线前2周）

| 渠道 | 目标人数 | 方式 |
|------|----------|------|
| 朋友圈定向邀请 | 200人 | KOL/核心用户内测 |
| 职场社群渗透 | 300人 | 脉脉/知乎/即刻 |
| 校园推广 | 200人 | 大学职场社团 |
| 内部员工 | 100人 | 公司同事内测 |

**内测激励**:
- 前100名用户获得「创始用户」终身徽章
- 内测期间无限次检测（正式上线后每日限3次）
- 邀请好友额外获得检测次数

### 1.2 内测机制

```
Day 1-3: 核心功能测试（检测/报告/分享）
Day 4-7: 社交功能测试（圈子/互动）
Day 8-14: 全功能开放，收集反馈
```

## 三、增长策略

### 3.1 分享裂变

**分享场景**:
- 检测完成 → 生成海报 → 分享朋友圈
- 打卡完成 → 生成对比 → 分享圈子
- 获得成就 → 生成徽章 → 分享炫耀

**邀请机制**:
- 每邀请1人 → 获得3次额外检测
- 被邀请人完成首次检测 → 双方获得「去味先锋」徽章
- 邀请排行榜 → 周榜Top10获得周边礼品

### 3.2 社交传播路径

```
用户A检测完成 → 分享海报到朋友圈
    ↓
用户B看到 → 扫码进入小程序
    ↓
用户B完成首次检测 → 建立邀请关系
    ↓
用户B分享 → 用户C进入...
```

### 3.3 渠道投放

| 层级 | 定义 | 运营策略 |
